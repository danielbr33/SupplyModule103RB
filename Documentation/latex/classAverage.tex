\subsubsection*{template$<$typename T, uint8\+\_\+t s$>$\newline
class Average$<$ T, s $>$}

Moving average. Class for objects which gather specified data and calculate moving average. Class use templates and it need to be declared with\+: ~\newline
 {\bfseries{typename T}} -\/ type of data to be calculted ~\newline
 {\bfseries{uint8\+\_\+t s}} -\/ fixed maximal number of samples take into average
