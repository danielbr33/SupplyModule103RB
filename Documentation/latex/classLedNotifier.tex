\doxysubsection{Detailed Description}
Tool intended for led\textquotesingle{}s, it provide typical behaviours for notifier -\/ O\+N/\+O\+FF or blink. To blink mechanism it use \mbox{\hyperlink{classTimeTool}{Time\+Tool()}} global class object to calculate time between toggles. To blink use \mbox{\hyperlink{classLedNotifier_a9f0c00eb5158c58b1d1f4870e892d589}{blink()}} function. \mbox{\hyperlink{classLedNotifier_a8fe6664b1157f3e86f0960ce985c7280}{blink\+\_\+process()}} could be used only by manager task.

Led blinking could be stopped by using \mbox{\hyperlink{classLedNotifier_ac8ee5ac9ad4666a1034f79f7dcdf4c4a}{on()}} / \mbox{\hyperlink{classLedNotifier_ab8e5a76adf9204775f52bf3a4a020872}{off()}} method
