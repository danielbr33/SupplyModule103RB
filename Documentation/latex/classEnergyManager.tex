\doxysubsection{Detailed Description}
Task class to manage branches energy measurements.
\begin{DoxyItemize}
\item Hall Sensor measurments (\mbox{\hyperlink{classHallSensor}{Hall\+Sensor()}})
\item Voltage on each section measurments (\mbox{\hyperlink{classSupplyBranch}{Supply\+Branch()}})
\item Manager of all section switches (\mbox{\hyperlink{classSectionSwitch}{Section\+Switch()}})
\item Temperature (maybe) -\/ because of sensor available in same A\+DC 
\end{DoxyItemize}
